# Ses Tanıma Uygulaması

<img src="image.png">

## Türkçe

Bu proje, Python ve FastAPI kullanarak ses tanıma yapan bir web uygulamasıdır. Kullanıcılar ses tanıma sonuçlarını gerçek zamanlı olarak görebilirler.

### Gereksinimler

- Python 3.7 veya üstü
- Pip (Python paket yöneticisi)

### Kurulum

1. Bu projeyi klonlayın veya indirin.
    ```bash
    git clone https://github.com/reburak/speech-recognition-app.git
    cd speech-recognition-app
    ```

2. Gerekli Python paketlerini yükleyin.
    ```bash
    pip install -r requirements.txt
    ```

### Kullanım

1. Backend sunucusunu başlatın.
    ```bash
    python test.py
    ```

2. HTML dosyasını yerel bir sunucuda açın.
    ```bash
    # Python 3 için
    python -m http.server 8080

    # veya
    npx serve
    ```

3. Tarayıcınızda `http://localhost:8080` adresine gidin ve uygulamayı kullanmaya başlayın.

## English

This project is a web application that performs speech recognition using Python and FastAPI. Users can see the speech recognition results in real-time.

### Requirements

- Python 3.7 or above
- Pip (Python package manager)

### Installation

1. Clone or download this project.
    ```bash
    git clone https://github.com/reburak/speech-recognition-app.git
    cd speech-recognition-app
    ```

2. Install the required Python packages.
    ```bash
    pip install -r requirements.txt
    ```

### Usage

1. Start the backend server.
    ```bash
    python app.py
    ```

2. Open the HTML file on a local server.
    ```bash
    # For Python 3
    python -m http.server 8080

    # or
    npx serve
    ```

3. Go to `http://localhost:8080` in your browser and start using the application.
